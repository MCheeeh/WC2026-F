## 🔧 **Comment utiliser ?**

### 1️⃣ **Obtenir une clé API gratuite**

👉 **Étape 1** : Rendez-vous sur [https://www.api-football.com](https://www.api-football.com)

👉 **Étape 2** : Cliquez sur **"Register"** (inscription gratuite)

👉 **Étape 3** : Remplissez le formulaire avec votre email

👉 **Étape 4** : Confirmez votre email (vous recevrez un lien de confirmation)

👉 **Étape 5** : Connectez-vous à votre compte

👉 **Étape 6** : Allez dans **"My Account"** → **"API Token"**

👉 **Étape 7** : Copiez votre clé API (elle commence par `xxxxxxxxxxxxxxxxxxxxxxxxxxxx`)
